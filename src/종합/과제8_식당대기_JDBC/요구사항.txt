종합과제8 요구사항_250723(수)
주제
연락처 와 인원수를 처리하는 대기번호프로그램 구축

기능
1. 대기 번호 등록 기능
   조건 : 대기번호는 자동부여 되며 연락처 와 인원수만 입력받습니다.

2. 대기 현황 전체 조회 기능
   조건 : 대기번호,연락처,인원수 모든 정보를 전체 출력합니다.

3. 특정 대기 취소
   조건 : 대기번호 만으로 삭제 처리 합니다.

4. 특정 대기 정보 수정
   조건 : 인원수만 수정 합니다.

기술스택
JAVA  , DATABASE

MVC 패턴
boardService10 구조와 동일하게 구현
